# Fluxus Quotidianus

**Fluxus Quotidianus** (Latin for **"Daily Flow"**) is a personal project inspired by the chaotic nature of everyday life. It aims to help users organize their daily routines, manage tasks efficiently, and cultivate a smoother, more intentional flow through each day with a simple and intuitive experience.

This project is currently a work in progress, and I am continuously developing and refining it by adding new features, improving usability, and learning throughout the journey.

Every update brings me one step closer to creating a tool that helps transform everyday chaos into a more balanced and purposeful daily flow.
